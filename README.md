# jit-aiml-chatbot
 simple chatbot using Artificial Intelligence Mark Up language using pythin

### clone this repo

**install python**

using cmd change the dir to cloned dir

TYPE **set FLASK_APP=chatbot.py**
**flask run**

local host will be started 
enter the IP address in your url

***use the chatbot***
